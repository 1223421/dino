#Dino
![screenshots](http://i.imgur.com/xIKPEFF.png)

##Install

###Dependencies
    glib2
    gtk3 (≥ 3.22)
    gpgme (For the OpenPGP plugin)
    libgee-0.8 (≥ 0.10)
    libnotify
    openssl (For the OMEMO plugin)
    sqlite3

**Build-time dependencies**

    cmake
    gcc
    ninja(-build) (recommend)
    valac (≥ 0.30)

###Build
    ./configure
    make
    build/dino
